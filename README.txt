
- 기능
1. 전체 채팅
2. 귓속말 채팅

- 역할
1) 클라이언트 (MsgClient)
사용자가 채팅 모드, 받는 이의 이름, 본문 내용을 콘솔로 입력하면
포트 8889 로 해당 메시지를 보냅니다.

메인서버, 중계서버와 달리 한 번 입력이 끝나면 프로그램이 종료됩니다.

2) 메인서버 (MsgServer)
채팅 참여자의 이름, ip 주소를 Map 으로 보관하고 있습니다.

포트 8889 로부터 클라이언트가 보낸 메시지를 받습니다.

클라이언트에게 받은 메시지를 메인서버 자신에게 출력합니다.
그 다음 메시지를 보기 좋게 편집하여 중계서버(MsgListener, 8888포트) 로 전달합니다.

3) 중계서버 (MsgListener)
포트 8888 에서 서버로부터 받은 메시지를 출력합니다.
